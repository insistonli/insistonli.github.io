2020-03-18 JavaScript JavaScript 0 评论 字数统计: 140(字) 阅读时长: 1(分)

e.clientX !

遇到类似拖拽，滑动等效果时，总是有点分不清楚 判断鼠标点击事件时的鼠标距离和作用div的距离，总结一下

对于点击事件源e来说

  • clientX，clientY
    点击位置距离当前可视区域的x，y坐标

  • pageX,pageY
    对于整个页面来说，包括了被卷去的body部分的长度

  • screenX，screenY
    点击位置距离当前电脑屏幕边界的x，y坐标

  • offsetX，offsetY
    相对于带有定位的父盒子的x，y坐标
